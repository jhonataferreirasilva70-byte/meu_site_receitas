Temperos do Brasil

Bem-vindo ao Temperos do Brasil!

Este site foi criado para apresentar receitas típicas das cinco regiões do Brasil: Norte, Nordeste, Centro-Oeste, Sudeste e Sul.

Aqui você pode pesquisar receitas, filtrar por região, salvar suas receitas favoritas e assistir vídeos para aprender o preparo de cada prato.

Explore os sabores, ingredientes e tradições da culinária brasileira.

Tecnologias utilizadas

HTML Usado para criar a estrutura das páginas, receitas, botões e janelas do site.
CSS Usado para criar cores, fontes, animações, modo claro e escuro e o visual do site.
JavaScript Usado para fazer os botões funcionarem, pesquisar receitas, favoritos, perfil, login e tema escuro.
LocalStorage Usado para salvar dados no navegador, como conta, senha, foto do perfil, favoritos e tema escolhido.
Font Awesome Usado para os ícones do site, como coração, perfil, lua, sol, lupa e informações.
Site desenvolvido para apresentar a diversidade da culinária brasileira 🇧🇷
